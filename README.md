# in-restful ![构建状态](https://travis-ci.org/gyk001/beetl2.0.svg)

http://www.ijson.com/

- 封装restful请求,支持POST,GET,PUT,DELETE
- 支持json及xml数据格式
- 单一依赖,支持多个不同服务地址
- 支持配置中心依赖下载

Maven 地址

    <dependency>
      <groupId>com.ijson</groupId>
      <artifactId>in-rest-proxy</artifactId>
      <version>1.0</version>
    </dependency>
